📋 How to Use This Tool

  • Place a reference object of known size in the scene (e.g., a 30cm ruler)
  • Enter the reference object's actual size in centimeters
  • Click "Calibrate with Reference" while showing the reference object
  • Select measurement type and click "Take Measurement"
  • For best results, ensure good lighting and clear visibility
  • Keep the camera steady during measurements
